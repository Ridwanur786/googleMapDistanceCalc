# googleMapDistanceCalc
calculating Distance Between Two Points Using google Maps Javascript API with auto Complete Search Input.

Few easy steps to make it work

1. get api key from google cloud consle
2. clone/download the code from the repo
3. enable google maps api libraries like geolocation, distance matrix and places .
4. put the api key on YOUR API KEY Section in the google api script tag at the bottom of the code.
5. don't forget to add &libraries=YOUR libraries after api key

and finally run this code in your local machine
